# nordic-eink-demo
Connect eink module to nordic dev board

## HW connections
(order as on 10040 dk pin out)
Pin     Connecton   Colour
P0.27   busy        purple
P0.26   Rst         white
P0.02   DC          Green
GND                 black
P0.25   CS          orange
P0.24   clk         yellow
P0.24   Din         blue
